# DSA210 Final Project

## Description
This project compares the rate of growth between 2 semesters, Fall and Spring, where the Fall had workouts done inconsitently at random times, while the Spring semester had workouts consistently in the morning.

## How to reproduce the analysis
1. Clone the repository to your local machine
2. Ensure you have python installed
3. Install the required dependencies mentioned in the "requirements.txt" by running "pip install -r requirements.txt"
4. Open the Jupyter Notebook ("EDA.ipynb") in your environment
5. Run all the cells one by one from top to bottom
